# Voyant
Digital passport to track all the countries you've visited, paired with an interactive map. 

### Tech Stack:
**frontend**: React, MUI, Clerk (auth), Mapbox (map)  
**backend**: Django (DB), Express (Img upload), Roboflow (labeling), S3 (Img store)
